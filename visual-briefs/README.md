# Visual Briefs

## Purpose

This folder stores controlled visual briefs before image generation or art commissioning.

A visual brief is the source-of-truth layer for an image request. It records what is known, what must be preserved, what remains unresolved, and how references should be used.

## Subfolders

- `characters/` — PC, NPC, crew, and variant character briefs.
- `locations/` — ship, island, port, interior, and scene-backdrop briefs.
- `maps/` — truth-first tactical map briefs.

## Rule

Do not generate or request a major visual asset without a brief unless the GM explicitly asks for a one-off sketch or experiment.
