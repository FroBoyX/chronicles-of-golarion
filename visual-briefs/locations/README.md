# Location Visual Briefs

Use this folder for location, ship, interior, exterior, and environmental scene briefs.

Location briefs should be created before establishing shots, interiors, approach views, scenic backdrops, or location variants.

Read first:

- `design/visual-output-method.md`
- `design/visual-style-guide.md`
- relevant ship/location/chronology files
- relevant encounter or AP scaffold notes
- `tracking/image-output-index.md`

Location art should preserve atmosphere and known landmarks. If exact geometry matters, route the task to `visual-briefs/maps/` instead.
