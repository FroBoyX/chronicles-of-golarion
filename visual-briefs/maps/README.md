# Map Visual Briefs

Use this folder for battle-map and tactical layout briefs.

Battle-map briefs are truth-first. Record tactical requirements before writing an image prompt.

Read first:

- `design/visual-output-method.md`
- `design/visual-style-guide.md`
- relevant map notes and encounter files
- relevant AP scaffold or GM campaign encounter data
- relevant chronology/state files
- `tracking/image-output-index.md`

Map briefs should distinguish player-safe and GM-only versions.

Required map concerns:

- dimensions or relative scale;
- grid/no-grid requirements;
- entrances and exits;
- terrain and traversal;
- elevation;
- cover and hazards;
- lighting/visibility;
- enemy/object placement if GM-only;
- VTT or print notes if needed.
