# Character Visual Briefs

Use this folder for PC, NPC, and crew visual briefs.

Character briefs should be created before portrait, full-body, crew portrait, expression, injury-state, outfit, or action-pose image generation.

Read first:

- `design/visual-output-method.md`
- `design/visual-style-guide.md`
- the relevant PC/NPC dossier
- relevant player discovery or current-state notes
- `tracking/image-output-index.md`

Character art should preserve identity, current state, and approved visual anchors. Do not resolve uncertain physical traits by convenience.
