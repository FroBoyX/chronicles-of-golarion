# Art Archive

## Purpose

This folder is reserved for campaign visual assets or metadata files for generated/commissioned art.

Large binary image files should only be committed here when the GM intentionally wants them stored in the repository. Otherwise, track asset links, filenames, prompt records, or external storage notes through `tracking/image-output-index.md` and the relevant visual brief.

## Subfolders

- `characters/` — character asset notes or stored character images.
- `locations/` — location asset notes or stored location images.
- `maps/` — map asset notes or stored map images.

## Rule

The asset index is the control record. Files here should not be treated as approved visual canon unless the index marks them as such.
