## Purpose

This file tracks the living narrative beats that should guide Mirei's player discovery chat.

It is not a replacement for `characters/pcs/mirei.md`. It is not final canon by itself. It records player-confirmed truth, uncertain memory, and GM-check items so future question batches build on prior answers instead of restarting.

## Handling Rules

- Treat player answers as **player-confirmed Mirei truth** unless they conflict with established evidence or are marked uncertain.
- Do not upgrade uncertain memory to canon without GM review.
- Keep Mirei's destiny, faith, and emotional interpretation player-led.
- Use this file to choose future questions.
- After each discovery batch, update the relevant beat rows or add new rows.
- If the player chat cannot write to the repo, output a GM Archive Handoff block for later update.

## Beat Status Values

- **Confirmed by player** — player has answered directly.
- **Open** — useful question remains unanswered.
- **Partial** — some player answer exists, but follow-up is still needed.
- **Needs GM check** — answer affects canon, chronology, or hidden state.
- **Archive integrated** — confirmed beat has been copied into a dossier, relationship file, or packet.
- **Retired** — no longer useful or superseded.
