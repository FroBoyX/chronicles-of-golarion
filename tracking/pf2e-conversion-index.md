# PF2e Conversion Index

## Document Status

- Type: Conversion tracking document
- Maturity: Working index
- Scope: PF1 to PF2e conversion targets, candidates, approvals, rejected paths, and open conversion risks

## Purpose

This file tracks possible Pathfinder 2e relaunch mechanics without overwriting Pathfinder 1e historical evidence.

PF2e conversion status does not change campaign canon. It only tracks possible or approved mechanics for future play.

## Status Values

- **Not started** — identified target, no audit yet.
- **Needs source review** — PF1 evidence or character truth not stable enough.
- **Draft audit** — conversion brief started.
- **Candidate proposed** — one or more PF2e paths proposed.
- **Player review needed** — PC conversion requires player review.
- **GM review needed** — awaiting GM mechanical/canon approval.
- **Approved relaunch mechanics** — adopted for future PF2e play.
- **Rejected** — conversion path rejected.
- **Superseded** — replaced by later conversion candidate.
