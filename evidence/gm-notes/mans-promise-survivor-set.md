- Type: GM confirmation / reconstruction note
- Maturity: Current GM truth

## GM Confirmation

The campaign has **six known tracked Man's Promise survivors**.

The current archive survivor set remains:

1. Dahrehn Numus
2. Ebony Saable
3. Goyle Onórach
4. Kahina Bukekayo
5. Merchiel V'giatore
6. Nor'gal Daraa

The later GM paste listed five names while saying six survivors. The missing sixth in that pasted list is **Kahina Bukekayo**, who is already part of the confirmed survivor set and is currently aboard the Man's Promise interacting with Sho.
