# PF2e Ship and Nautical Conversions

Use this folder for Pathfinder 2e ship, crew, command, naval travel, and nautical subsystem conversion briefs.

Ship conversion should preserve:

- ship identity;
- current ship state;
- crew viability;
- command legitimacy;
- social pressure aboard ship.

Do not reduce the campaign's shipboard politics to mechanics only. Crew relationships and authority remain core campaign systems.
