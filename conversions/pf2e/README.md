# PF2e Conversions

## Purpose

This folder stores Pathfinder 1e to Pathfinder 2e conversion audits and approved relaunch mechanics for Chronicles of Golarion.

Read first:

- `design/pf2e-conversion-method.md`
- `handoffs/pf2e-conversion-handoff.md`
- `tracking/pf2e-conversion-index.md`

## Subfolders

- `pcs/` — player character conversion briefs.
- `npcs/` — NPC conversion briefs or stat-use decisions.
- `encounters/` — encounter conversion briefs.
- `items/` — item and treasure conversion briefs.
- `ships/` — ship, crew, and nautical subsystem conversion briefs.

## Rule

PF2e conversion is a relaunch mechanics layer. It should not overwrite PF1 historical evidence or imply old outcomes should change.
