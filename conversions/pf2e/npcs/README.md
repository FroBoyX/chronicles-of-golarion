# PF2e NPC Conversions

Use this folder for Pathfinder 2e conversion briefs for NPCs.

Most NPCs do not need full stat blocks. Decide whether the NPC needs:

- narrative-only treatment;
- a role/DC note;
- an existing PF2e creature or NPC stat block;
- a custom stat block.

Use full mechanics only when the NPC is expected to matter tactically.
