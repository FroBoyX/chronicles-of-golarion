# Conversions

## Purpose

This folder stores mechanics-conversion work that supports possible future play without rewriting campaign history.

Conversion files are not raw evidence. They are derived mechanical proposals, audits, or approved relaunch mechanics.

## Current Conversion Streams

- `pf2e/` — Pathfinder 1e to Pathfinder 2e conversion audits and relaunch mechanics.

## Rule

Historical Pathfinder 1e sheets, rolls, and table outcomes remain evidence. Conversion documents should link back to those sources rather than replacing them.
